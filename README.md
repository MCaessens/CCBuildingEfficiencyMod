# Cookie Clicker Building Efficiency Mod
This mod of Cookie Clicker shows the efficiency of buying buildings.

### How to use?

You can use this mod by downloading the info.txt and the main.js file. And putting them into a folder in the mods folder of Cookie Clicker.
The mod folder can be found by right clicking on Cookie Clicker -> Manage -> Browse local files.

When you have done this step, a folder will open with multiple files in it. You should then open the resources folder -> app -> mods -> local.

When you are here, you can copy and paste the folder you made with the info.txt and the main.js file into the local folder.

Then you just go to options -> manage mods and the Building Efficiency mod should be in the list. Then you just click `Enable` and you should be done!

## THIS MOD DOES NOT DISABLE GETTING ACHIEVEMENTS.
